Course Title: Introduction to Web Development

Instructor: Jane Doe

Course Description

This course provides an introduction to web development. Students will learn the basics of HTML, CSS, and JavaScript, and how to create responsive and interactive web pages.

Course Objectives

  • Understand the structure and syntax of HTML
  • Learn how to style web pages using CSS
  • Develop interactive features using JavaScript
  • Build responsive web pages that work on various devices

Course Schedule

  • Week 1: Introduction to HTML
  • Week 2: Advanced HTML and Semantic Elements
  • Week 3: Introduction to CSS
  • Week 4: CSS Layout and Design
  • Week 5: Introduction to JavaScript
  • Week 6: JavaScript Events and DOM Manipulation
  • Week 7: Responsive Web Design
  • Week 8: Project Work and Final Assessment

Grading Policy

  • Assignments: 40%
  • Midterm Exam: 20%
  • Final Project: 30%
  • Class Participation: 10%

Textbooks and Resources

  • HTML and CSS: Design and Build Websites by Jon Duckett
  • JavaScript and JQuery: Interactive Front-End Web Development by Jon Duckett
